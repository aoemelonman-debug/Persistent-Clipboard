# 🧠 Persistent Clipboard

A simple web-based clipboard manager that stores unlimited text snippets and keeps them saved even after refreshing the page.

## 🚀 Features

- Add unlimited clipboard items
- Persistent storage using `localStorage`
- Copy items with one click
- Delete individual items
- Clear all items
- Works completely in the browser (no backend)

## 📦 How to Use

1. Download or clone this repository
2. Open `index.html` in your browser
3. Start adding text to your clipboard!

## 💡 Tech Used

- HTML
- CSS
- JavaScript (Vanilla)
- Browser localStorage API

## 📌 Why this exists

This project is useful for developers or users who often copy and reuse text snippets, links, or code.
